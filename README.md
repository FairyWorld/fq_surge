## 节点正则表达式过滤：
(A).*(B)             节点名既有 A又有 B 

(A)|(B)              节点名有 A 或者 B   

^((?!A).)*$或^(?!A).*   节点名不含有 A 

(?!.*(A)).*(B)         节点名不含有 A，同时含有 B

^(?!(.*(A|B).*))      节点名既不含有 A又不含有 B

### 更新频率
每天不定期更新，为了保证高可用性会定期吧ping不通的节点剔除，避免过多节点导致性能下降。
